
 📊 Sales & Delivery Dashboard

 📌 Overview

Developed an interactive Excel dashboard to monitor sales performance and delivery efficiency.

 ✨ Decription
 This project presents an interactive Sales and Delivery Dashboard built using Microsoft Excel.
It helps analyze sales performance, delivery timelines, and operational efficiency.

 📈 Key Features

* Sales trend analysis (last 13 weeks)
* Product popularity insights
* Customer purchase behavior analysis
* Delivery time & delay tracking
* Customer rating evaluation

🛠 Tools Used

* Microsoft Excel
* Pivot Tables
* Charts & Data Visualization

 🚀 How to Use

Download the Excel file and explore using filters and slicers.
